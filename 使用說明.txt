# 保健食品庫存管理系統 - 使用說明書

歡迎使用本庫存管理系統！本專案為您準備好了所有 23 張圖片的對應與精美網頁介面。

## 🚀 如何啟動系統 (推薦)

雙擊資料夾中的 `雙擊啟動系統.bat` 檔案即可！
系統會執行以下動作：
1. 自動開啟您的瀏覽器並進入 `http://localhost:8000`。
2. 啟動一個極輕量、安全的本地 Python 伺服器，負責自動將您編輯的內容保存至同目錄下的 `data.json` 中。
3. 關閉網頁時，只要在黑色的命令提示字元視窗按 `Ctrl + C`（或直接關閉視窗）即可關閉伺服器。

---

## 💻 方案 B：直接開網頁使用 (免執行任何程式)

如果您不想執行伺服器，也可以直接雙擊 `index.html` 檔案：
- **一樣可以檢視圖片與編輯！**
- 在此「單機模式」下，您輸入的所有資料會自動儲存於您電腦瀏覽器的「暫存區 (LocalStorage)」。
- **防遺失機制**：建議您填寫完畢後，點擊右上角的「匯出備份 (JSON)」下載備份檔案。下次開啟時若發現資料空白，只需點擊「匯入備份」選取該 JSON 檔即可瞬間復原。

---

## 🌟 網頁功能亮點

1. **圖片懸停放大**：滑鼠移到圖片縮圖上，會自動浮現中型尺寸的預覽。
2. **點擊大圖燈箱**：點擊圖片會開啟全螢幕燈箱。您可以：
   - 使用鍵盤左右方向鍵 `←` `→` 切換照片。
   - 點擊底部工具列進行「順時針/逆時針旋轉」、「放大」、「縮小」與「重設」。
3. **即時編輯 & 自動儲存**：
   - 表格中的任何欄位（補給品名稱、品牌、數量、備註等）點擊即可打字，修改後 1 秒鐘內系統會自動存檔。
   - 數字欄位（總瓶數、全新、已開）輸入「全新」與「已開」後，系統會幫您自動相加計算出「總瓶數」。
4. **卡片檢視 (Grid View)**：可隨時切換成卡片風格，像電商網站一樣精美，同樣支援即時修改。
5. **快速搜尋**：輸入任何文字，表格會即時篩選出相符的保健食品。
6. **統計看板**：頂部即時顯示您的「補給品總類」、「總瓶數」、「全新」與「已開」累計數量。
7. **匯出 Excel (CSV)**：填寫完成後可一鍵下載 CSV 表格，直接使用 Excel 開啟。
8. **護眼深色模式**：點擊右上角的「月亮/太陽」圖示可切換主題。

## 📂 檔案目錄結構

- `index.html` - 網頁骨架
- `style.css` - 網頁精美樣式與動畫
- `app.js` - 前端邏輯（搜尋、匯出、自動存檔）
- `server.py` - 本地 Python 伺服器
- `雙擊啟動系統.bat` - 一鍵開啟伺服器指令檔
- `data.json` - 自動生成的資料庫檔案 (您的資料都在這！)
- `使用說明.txt` - 本說明書
- 23張原始圖片檔
